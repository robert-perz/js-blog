## Installation

App runs at [@localhost:3000](`http://localhost:3000/`)

In project directory

Install all dependencies

```
npm install
```

Start to watch db file/s with JSON server

```
npx json-server --watch data/db.json --port 1309
```

In project directory

```
npm run start
```
